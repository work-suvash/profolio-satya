# **App Name**: PortfolioPro

## Core Features:

- Responsive Layout: A fully responsive design that adapts to different screen sizes (desktop, tablet, mobile).
- Hero Section: A dynamic hero section with a profile image, personal introduction, key stats, and social media links.
- Services Section: Highlight services offered through visually appealing cards with icons and descriptions.
- Projects Section with Filtering: Showcase completed projects in a grid layout with filtering options (e.g., All, Web development, App development, Newly added).
- Fullscreen Overlay Modal: Implement a fullscreen overlay modal for displaying project details (videos or images) with smooth zoom-in animation and close behavior.
- Contact Form: A contact form with input validation to gather user information and service inquiries.

## Style Guidelines:

- Primary color: Deep dark black (#000000) for the background to create a modern and professional feel.
- Accent color: Vibrant orange (#FFA500) to be used for headings, icons, and active states to draw attention.
- Headline and body font:  'Inter', a sans-serif, as it offers a clean and modern look suitable for both headings and body text.
- Use a consistent set of minimalist icons, with orange as the primary color, for services and project categories.
- Maintain consistent spacing and alignment throughout the website, using Flexbox and Grid for responsive layouts.
- Implement subtle transition effects for hovers and clicks to enhance user engagement.
